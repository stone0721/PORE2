### 解答

三层迷宫一体，既然知道了三个迷宫，我们可以先遍历找找所有的合法的坐标，即`valid(x,y)`的返回值为1，把这些点打印出来。

```py
def valid(x, y):
    if y % 3 == 0:
        return map1[x][y // 3] == '*'
    elif y % 3 == 1:
        return map2[64 - x][y // 3] == '*'
    elif y % 3 == 2:
        return map3[x][y // 3] == '*'
    return False

# 绘制有效点
for i in range(65):
    for j in range(30):
        if valid(i, j):
            ax.plot(j, i, 'k.')  # 绘制黑色点

```

由于`x`每次加减 2，`y`每次加减 1，画出的散点图还是太难看了，我们让`x`每次加减 1，删去字符串数组中索引为偶数的字符串，反正永远不会被用到，`x`的范围由`0 <= x <= 62`变为` 0 <= x <= 31`。

目标就变成了(28,89)
